导出备份

把当前所有数据导出成 JSON 文件，适合换浏览器、换电脑前先备份。

建议定期手动导出，浏览器清缓存后本地记录可能会消失。

导入恢复

支持两种方式：上传之前导出的 JSON 文件，或把 JSON 文本粘贴到下方再导入。

导入会覆盖当前页面数据，导入前建议先导出一份当前备份。

修复与清理

遇到忘记结束、误触开启等情况，可以先用轻量修复，而不是直接清空全部数据。

  • 会清除专注中的 active 状态。
  • 会清除三时段考勤里还没结束的 active 段。
  • 不会删除已经完成的历史记录。

危险操作

会清空这套工作台当前存储键下的全部数据。执行前请先导出备份。

此操作不可撤销，除非你之前已经导出过 JSON 备份。
